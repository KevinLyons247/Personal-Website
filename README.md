# Kevin Lyons - Personal Website

A modern, responsive personal website built with React and Node.js showcasing my portfolio, skills, and experience.

## 🚀 Technologies Used

- **Frontend**: React, HTML5, CSS3
- **Backend**: Node.js, Express
- **Styling**: Custom CSS with modern gradients and animations

## 📋 Features

- Responsive design that works on all devices
- Smooth scrolling navigation
- Modern UI with gradient backgrounds and animations
- Links to GitHub and LinkedIn prominently displayed
- Sections for Education, Skills, Experience, Projects, and Activities
- Contact information display

## 🛠️ Installation & Setup

### Prerequisites
- Node.js (v14 or higher)
- npm or yarn

### Installation Steps

1. **Install all dependencies** (root, client, and server):
   ```bash
   npm run install-all
   ```

2. **Run the development server** (runs both frontend and backend):
   ```bash
   npm run dev
   ```

   This will start:
   - React app on `http://localhost:3000`
   - Express server on `http://localhost:5000`

### Alternative: Run separately

**Start the backend server:**
```bash
npm run server
```

**Start the React frontend** (in a new terminal):
```bash
npm run client
```

## 📦 Build for Production

To create a production build:

```bash
npm run build
```

This will create an optimized build in the `client/build` directory.
